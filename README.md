# Password_Generator-Checker
This project generates strong, random passwords and evaluates their strength in real time. It checks length, uppercase/lowercase letters, numbers, and special characters to ensure maximum security. Simple, fast, and user-friendly—perfect for improving online account protection and learning about password security best practices.
🔐 Password Generator & Strength Checker

A powerful and secure Password Generator and Password Strength Checker built using Bash and C programming language.

This tool allows users to:

Generate strong random passwords

Generate passwords based on their name

Check password strength (Weak / Medium / Strong)

Ensure security using best password practices

🚀 Features

🔑 Generate strong random passwords

🧑 Generate password using your name

🔍 Check password strength

⚡ Fast execution (written in C)

🖥️ Works on Linux / Unix systems

🧠 Follows strong password rules (uppercase, lowercase, numbers, symbols)

🛠️ Tech Stack

Language 1: C

Language 2: Bash

Platform: Linux / Unix
